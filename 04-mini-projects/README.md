# Mini Projects

Hands-on projects that combine multiple topics. Each project ships with a sample dataset (CSV) and a step-by-step walkthrough that produces a working spreadsheet, report, or dashboard.

| # | Project | Skills Used | File |
|---|---------|-------------|------|
| 01 | Sales Dashboard | PivotTables, Charts, Lookups | [01-sales-dashboard/](01-sales-dashboard/README.md) |
| 02 | Personal Budget Tracker | Formulas, Conditional Formatting, Validation | [02-budget-tracker/](02-budget-tracker/README.md) |
| 03 | Survey Data Analysis | COUNTIF, Text/Logical Functions, Charts | [03-survey-analysis/](03-survey-analysis/README.md) |
| 04 | Inventory Manager | Lookups, Data Validation, IF alerts | [04-inventory-manager/](04-inventory-manager/README.md) |
| 05 | Data Cleaning Challenge | Text functions, Remove Duplicates, Power Query | [05-data-cleaning-challenge/](05-data-cleaning-challenge/README.md) |

Each project folder contains:
- A `README.md` with the goal, the dataset description, and a guided walkthrough.
- One or more `.csv` data files you can import into Excel to follow along.
